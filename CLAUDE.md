# Context

This project serves as a reference codebase for students in the Technical University of Applied Sciences Zurich (TBZ) HF program, specifically for the DevOps & Infrastructure module block.

**Purpose**  
Provides a realistic, hands-on learning environment where students can practice and validate solutions against course tasks spanning Day 01 through Day 20 of the curriculum.

**Key Characteristics**
- Real-world Flask eCommerce application
- Iterative development and testing workflow
- Clear separation of concerns (frontend, backend, data layers)
- CI/CD pipeline integration ready
- Deployment automation examples included

**Target Audience**  
Students completing the DevOps & Infrastructure module block at TBZ.

**Important Notes**
- Run the app-health-check-agent after each change to verify application health
- When updating a solution branch, propagate changes to the corresponding checkpoint branch
