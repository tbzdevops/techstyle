# Context

This project serves as a reference codebase for students in the Technical University of Applied Sciences Zurich (TBZ) HF program.

**Purpose**  
The codebase provides a realistic, hands-on environment where students can practice and validate their solutions against the module block tasks covered in the DevOps course.

**Key Characteristics**
- Real-world Flask eCommerce application
- Support for iterative development and testing
- Clear separation of concerns (frontend, backend, data)
- Designed for CI/CD pipeline integration
- Includes deployment automation examples

**Target Audience**  
Students completing the DevOps & Infrastructure module block at TBZ, working through tasks from Day 01 through Day 20 of the curriculum.
